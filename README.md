# Patient Position Monitoring

This project presents a computer vision-based approach for patient position monitoring in medical imaging and radiotherapy environments.

## Technologies
- Python
- Google Colab
- Detectron2
- Mask R-CNN
- Keypoint R-CNN
- OpenCV
- NumPy
- Pandas

## Outputs
- Centroid displacement
- Body area estimation
- Keypoint tracking
- Movement classification
